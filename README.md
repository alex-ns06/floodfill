# Flood Fill em Java

## Descrição

Este projeto implementa o algoritmo Flood Fill em Java. Ele serve para preencher uma área de uma imagem a partir de um ponto inicial, mudando a cor dos pixels conectados que possuem a mesma cor.

## Como funciona

O programa recebe uma imagem, escolhe um ponto inicial e altera a cor da região conectada a esse ponto para uma nova cor.

O algoritmo foi implementado de duas formas:

* Usando fila
* Usando pilha
